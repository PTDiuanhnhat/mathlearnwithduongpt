📊 Chủ đề 1: Hàm Số

1. Khái niệm hàm số

Định nghĩa: Hàm số f từ tập A đến tập B là quy tắc đặt tương ứng mỗi phần tử x ∈ A với duy nhất một phần tử y ∈ B

Ký hiệu: y = f(x) hoặc f: A → B

Biến số: x gọi là biến độc lập

Hàm số: y gọi là biến phụ thuộc

2. Tập xác định

Định nghĩa: Tập xác định D của hàm số y = f(x) là tập hợp tất cả các giá trị của x để f(x) có nghĩa

Các trường hợp:

• Hàm đa thức: D = ℝ

• Hàm phân thức: D = ℝ \ {nghiệm mẫu}

• Hàm căn: D = {x | biểu thức dưới căn ≥ 0}

3. Đồ thị hàm số

Định nghĩa: Đồ thị hàm số y = f(x) là tập hợp các điểm M(x; f(x)) trong mặt phẳng tọa độ

Cách vẽ đồ thị:

1. Tìm tập xác định

2. Lập bảng giá trị

3. Vẽ các điểm và nối lại

4. Tính chất hàm số

Tính đơn điệu:

• Đồng biến: x₁ < x₂ ⇒ f(x₁) < f(x₂)

• Nghịch biến: x₁ < x₂ ⇒ f(x₁) > f(x₂)

• Tính chẵn lẻ: f(-x) = f(x) (chẵn), f(-x) = -f(x) (lẻ)

💡 Ví dụ minh họa

Ví dụ 1: Tìm TXĐ của y = √(x - 1)

Giải: x - 1 ≥ 0 ⇒ x ≥ 1. Vậy D = [1; +∞)

Ví dụ 2: Tìm TXĐ của y = 1/(x² - 4)

Giải: x² - 4 ≠ 0 ⇒ x ≠ ±2. Vậy D = ℝ \ {-2; 2}

📈 Chương 2: Hàm Số

1. Khái niệm hàm số

Định nghĩa: Hàm số f từ tập A đến tập B là quy tắc đặt tương ứng mỗi phần tử x ∈ A với duy nhất một phần tử y ∈ B

Ký hiệu: y = f(x) hoặc f: A → B

Tập xác định: D(f) = A

Tập giá trị: T(f) ⊆ B

2. Hàm số bậc nhất

Dạng tổng quát: y = ax + b (a ≠ 0)

Tập xác định: D = ℝ

Tính chất:

• Đồng biến khi a > 0

• Nghịch biến khi a < 0

• Đồ thị là đường thẳng

3. Hàm số bậc hai

Dạng tổng quát: y = ax² + bx + c (a ≠ 0)

Tập xác định: D = ℝ

Đỉnh parabol: x₀ = -b/2a

Trục đối xứng: x = -b/2a

Giá trị tại đỉnh: y₀ = -Δ/4a

4. Đồ thị hàm số

Hàm bậc nhất: Đường thẳng

Hàm bậc hai: Parabol

Cách vẽ:

1. Lập bảng giá trị

2. Xác định các điểm đặc biệt

3. Vẽ đồ thị

💡 Ví dụ minh họa

Ví dụ 1: Cho y = 2x + 3. Tính f(1), f(-2)

Giải: f(1) = 2(1) + 3 = 5; f(-2) = 2(-2) + 3 = -1

Ví dụ 2: Tìm đỉnh parabol y = x² - 4x + 3

Giải: x₀ = 4/2 = 2; y₀ = 4 - 8 + 3 = -1. Đỉnh I(2; -1)

📉 Chủ đề 3: Dấu Của Tam Thức Bậc Hai

1. Tam thức bậc hai

Định nghĩa: Tam thức bậc hai là biểu thức có dạng f(x) = ax² + bx + c (a ≠ 0)

Biệt thức: Δ = b² - 4ac

Nghiệm của tam thức:

• Δ > 0: Có 2 nghiệm phân biệt

• Δ = 0: Có 1 nghiệm kép

• Δ < 0: Vô nghiệm

2. Dấu tam thức bậc hai

Quy tắc xét dấu: Dấu của tam thức phụ thuộc vào hệ số a và biệt thức Δ

Khi a > 0:

• f(x) > 0 ngoài khoảng nghiệm

• f(x) < 0 trong khoảng nghiệm

Khi a < 0: Ngược lại

3. Bảng xét dấu

Trường hợp Δ > 0: Có 2 nghiệm x₁ < x₂

Bảng xét dấu (a > 0):

(-∞; x₁): f(x) cùng dấu a

(x₁; x₂): f(x) trái dấu a

(x₂; +∞): f(x) cùng dấu a

4. Ứng dụng

Giải bất phương trình:

• f(x) > 0: Lấy khoảng f(x) dương

• f(x) < 0: Lấy khoảng f(x) âm

• f(x) ≥ 0: Thêm nghiệm vào tập nghiệm

• f(x) ≤ 0: Thêm nghiệm vào tập nghiệm

💡 Ví dụ minh họa

Ví dụ 1: Xét dấu f(x) = x² - 5x + 6

Giải: Δ = 1 > 0, x₁ = 2, x₂ = 3
f(x) > 0 khi x ∈ (-∞;2) ∪ (3;+∞)

Ví dụ 2: Giải BPT x² - 4x + 3 < 0

Giải: Nghiệm x₁ = 1, x₂ = 3
Nghiệm BPT: x ∈ (1; 3)

🧮 Chủ đề 4: Phương Trình Quy Về Phương Trình Bậc Hai

1. PT chứa ẩn ở mẫu

Dạng: P(x)/Q(x) = R(x)/S(x) hoặc P(x)/Q(x) = a

Cách giải:

Bước 1: Tìm ĐKXĐ: Q(x) ≠ 0, S(x) ≠ 0

Bước 2: Quy đồng mẫu và khử mẫu

Bước 3: Giải PT thu được

Bước 4: Kiểm tra ĐKXĐ

2. PT chứa căn thức

Dạng: √f(x) = g(x) hoặc √f(x) = √g(x)

Phương pháp:

• Tìm ĐKXĐ: f(x) ≥ 0

• Bình phương hai vế

• Giải PT bậc hai thu được

• Kiểm tra nghiệm

3. PT tích

Dạng: A(x) · B(x) = 0

Nguyên lý:

A(x) · B(x) = 0 ⇔ A(x) = 0 hoặc B(x) = 0

Cách giải:

• Giải A(x) = 0

• Giải B(x) = 0

• Hợp nghiệm

4. PT đặt ẩn phụ

Dạng: PT có thể đưa về dạng at² + bt + c = 0

Các dạng thường gặp:

• ax⁴ + bx² + c = 0 (đặt t = x²)

• a(x + 1/x)² + b(x + 1/x) + c = 0

• PT chứa √x và x

💡 Ví dụ minh họa

Ví dụ 1: Giải (x-1)/(x+2) = 3

Giải: ĐKXĐ: x ≠ -2
x - 1 = 3(x + 2) ⇒ x = -3.5

Ví dụ 2: Giải x⁴ - 5x² + 4 = 0

Giải: Đặt t = x² (t ≥ 0)
t² - 5t + 4 = 0 ⇒ t = 1, t = 4
⇒ x = ±1, x = ±2
